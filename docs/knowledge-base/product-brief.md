# Product Brief

## Product

Bliss is an offline-first Android game built in Godot for helping nonverbal autistic children learn Blissymbolics.

## Audience

Primary audience:
- nonverbal autistic children

Initial real-world test:
- one 10-year-old child

## V1 Product Position

V1 is a learning game first.

V1 is not yet:
- a full AAC communication tool
- a free-form sentence builder
- a parent-heavy setup experience

## V1 Learning Goals

- symbol recognition
- picture to symbol matching
- symbol to picture matching

## Child Flow Constraints

Required:
- drag-and-drop interaction
- picture and symbol paired together at first
- short 3 to 5 minute sessions
- calm presentation
- minimal text
- sound effects only
- rewards mainly at session end

Avoid:
- timers
- failure screens
- distracting sounds
- music in V1

## Reward Style

- stars
- gentle praise
- small animations
- unlocks based on session completion

## Parent Side

Include a hidden parent-facing progress screen.

V1 parent metrics:
- symbols learned
- categories mastered
